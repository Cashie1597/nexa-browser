# Brand

Nexa Browser is designed to feel clean, fast, private, and intelligent.

The brand should feel modern and premium without copying any existing browser identity.

## Brand Direction

Nexa should feel:

- Minimal
- Fast
- Private
- Intelligent
- Calm
- Premium
- Mobile-first

## Inspiration

Nexa is inspired by the best ideas from modern browsers:

- Chrome-inspired simplicity
- Brave-inspired privacy
- Yandex-inspired AI assistance
- Orion-inspired extension flexibility

Nexa does not copy logos, icons, names, source code, or proprietary designs from any browser.

## Visual Style

The visual style should use:

- Clean rounded shapes
- Soft shadows
- Simple icons
- Dark mode support
- Smooth animations
- Clear spacing
- Minimal clutter

## Colour Direction

Suggested colour direction:

- Deep navy or black background
- Electric blue accent
- Soft white text
- Subtle grey cards
- Optional purple AI accent

## Tone

The app should feel:

- Helpful
- Direct
- Secure
- Smart
- Lightweight

## Tagline Ideas

- A smarter browser for iPhone and iPad.
- Browse faster. Think smarter.
- Private browsing with built-in AI.
- Your AI-powered browser for iOS and iPadOS.
- Clean browsing. Smart tools. Better focus.

## Naming Note

Nexa Browser is a working project name. Before public launch, the name should be checked for trademark conflicts.
