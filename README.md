# Nexa Browser
Nexa Browser is a SwiftUI browser concept for iOS and iPadOS inspired by the best ideas from Chrome, Brave, Yandex, and Orion.

It uses WKWebView for browsing and focuses on privacy, AI assistance, clean design, and modular extension-style tools.

## Vision

Nexa aims to be a modern mobile browser with:

- Chrome-inspired simplicity
- Brave-inspired privacy tools
- Yandex-inspired AI assistance
- Orion-inspired extension-style flexibility

Nexa does not copy or use branding, icons, source code, or proprietary assets from any existing browser.

## Planned Features

### Browser Core

- SwiftUI iPhone and iPad interface
- WKWebView browsing
- Search/address bar
- Tabs
- Private browsing
- Bookmarks
- History
- Search engine selector
- iPad sidebar layout

### Privacy

- Shield button
- Content blocker
- Tracker blocking
- Per-site privacy settings
- HTTPS-first mode concept
- Cookie controls

### AI Assistant

- AI sidebar
- Summarise current page
- Ask questions about the current page
- Explain selected text
- Translate selected text
- Rewrite selected text
- Smart search prompts

### Extensions

Nexa uses a curated extension-style system rather than full Chrome extension support.

Planned extension types:

- Built-in native extensions
- JavaScript userscripts
- Limited WebExtension-style manifest support

Planned extensions:

- AI Summariser
- Reader Mode
- Dark Reader Lite
- Focus Mode
- Page Notes
- Content Blocker
- Translate Page

## Tech Stack

- Swift
- SwiftUI
- WebKit / WKWebView
- JavaScript userscripts
- OpenAI API or OpenRouter API
