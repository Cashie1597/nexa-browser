# Contributing

Thanks for your interest in contributing to Nexa Browser.

Nexa Browser is currently in the planning stage. Contributions are welcome for documentation, feature ideas, SwiftUI structure, privacy tools, AI features, and extension-system concepts.

## Ways to Contribute

You can help by:

- Improving documentation
- Suggesting features
- Reporting issues
- Designing UI mockups
- Writing SwiftUI components
- Improving privacy ideas
- Improving AI assistant ideas
- Improving the extension system

## Project Goals

Nexa Browser aims to be:

- Clean
- Fast
- Privacy-first
- AI-assisted
- iPhone and iPad friendly
- Modular and easy to understand

## Contribution Guidelines

Please keep contributions:

- Clear
- Respectful
- Useful
- Well explained
- Aligned with the project vision

## Pull Requests

When opening a pull request, please include:

- What changed
- Why it changed
- Screenshots if it affects UI
- Any limitations or future work

## Issues

When opening an issue, please include:

- A clear title
- A short description
- Steps to reproduce, if reporting a bug
- Expected result
- Actual result
- Device or iOS/iPadOS version, if relevant

## Important Notes

Nexa Browser does not copy branding, icons, proprietary features, or source code from Chrome, Brave, Yandex, Orion, or any other browser.

The project is inspired by modern browser ideas, but it should remain original.
